Обычно в Java каждый класс объявлен в отдельном файле. В файле может быть несколько классов, но публичный -- только один.
У классов и их содержимого есть область видимости, которая задаётся модификаторами доступа(public, protected и т.д.).
При желании мы можем импортировать один класс в другой и использовать публичные поля и методы импортированного класса.
Помимо импорта, мы можем расширить класс, с помощью наследования и добавить туда новое поведение.

Классы можно объединять в пакеты. Пакет задаёт пространство имён. И внутри одного пакета не может быть двух классов с одинаковым именем.
При импорте мы указываем имя пакета, к которому принадлежит класс, это позволяет импортировать классы с одним названием из разных пакетов.
Пакеты можно представить в виде директорий, а классы в виде файлов.

В Java 9 появилась концепция модулей. Модуль является более высоким уровнем абстракции, нежели пакет. 
Модуль содержит в себе множество пакетов. Благодаря модулям мы можем ограничивать доступ программы к определённым пакетам.
Также мы можем указывать другие модули от которых зависит наш модуль.

Всё это позволяет разбивать программу на части, которые затем можно переиспользовать в других местах.
