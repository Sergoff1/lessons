#### Предварительное описание системы и наиболее общие сущности игры три-в-ряд
1. **Поле**(сетка). Содержит в себе игровые элементы, с которыми взаимодействует пользователь. Может заполняться элементами, целиком обновляться, проверять состояние(есть ли возможность сделать ход).
2. **Элемент**. Предмет, располагаемый на поле. Их группа может составлять комбинации из одного типа, которые должны исчезать с поля. Игрок взаимодействует с элементами, путём перестановки элемента с соседним. Есть особый вариант элемента, получаемый в виде бонуса, который можно активировать без перестановки. В результате активации, с поля будут удалена часть элементов. 
3. **Статистика**. Отвечает за подсчёт очков, которые получает пользователь в результате составления комбинаций. Также ведёт историю ходов.
4. **Комбинация**. Группа элементов одного типа, находящихся рядом друг с другом, которая должна исчезнуть в начале хода. Даёт очки и бонусные элементы, особые сущности на поле, которые убирают другие элементы, при их активации. 
5. **Интерфейс**. Отвечает за отрисовку поля и элементов, отображение статистики, получение команд от пользователя.
6. **Ход**. Действие игрока. Перестановка элементов или активация специального элемента. В результате совершения хода выполняется обработка комбинаций, заполнение поля новыми элементами и проверка его состояния.
