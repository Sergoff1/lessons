Допустим у нас есть сайт, где люди могут публиковать свои статьи.  
Там может быть такая структура таблиц:  
Таблица articles: 
- article_id, 
- author_id, 
- title, 
- content, 
- published_date

Таблица authors: 
- author_id, 
- username,
- avatar_url

Для отображения списка статей с именем автора на главной странице нужен JOIN для каждой загрузки страницы.

Для ускорения запроса на получение списка статей с авторами, мы можем денормализовать таблицу articles:
Новая таблица articles:
- article_id,
- title,
- content,
- published_date
- username,
- avatar_url

Теперь получение списка статей с их авторами обходится нам в один простой запрос без джоинов.
